"메모아르 덩크리에" 사이트는 고급 스크롤 애니메이션을 사용하여 시각적으로 매력적인 경험을 제공합니다. 이 애니메이션은 주로 GSAP (GreenSock Animation Platform)와 ScrollMagic 같은 라이브러리를 사용하여 구현됩니다. 이러한 애니메이션은 스크롤 시 요소들이 부드럽게 떠다니는 효과를 줍니다.

주요 애니메이션 효과:
GSAP를 통한 애니메이션: GSAP는 텍스트, 이미지, 배경 등의 요소를 애니메이션화하는 데 사용됩니다. 이는 부드럽고 성능이 뛰어난 애니메이션을 제공하여 매끄러운 사용자 경험을 만듭니다.

ScrollMagic과의 결합: ScrollMagic은 스크롤 위치에 따라 애니메이션을 트리거하고 조정할 수 있게 해줍니다. 이를 통해 사용자가 페이지를 스크롤할 때 요소들이 부드럽게 이동하고 나타나는 효과를 만들 수 있습니다.

코드 예시:
다음은 GSAP와 ScrollMagic을 사용하여 스크롤 애니메이션을 구현하는 예제입니다.

html
코드 복사
<!DOCTYPE html>
<html lang="ko">
<head>
    <meta charset="UTF-8">
    <title>스크롤 애니메이션</title>
    <style>
        body { height: 200vh; }
        .floating { position: relative; width: 100px; height: 100px; background-color: red; margin-top: 100px; }
    </style>
</head>
<body>
    <div class="floating"></div>

    <!-- GSAP 및 ScrollMagic 라이브러리 추가 -->
    <script src="https://cdnjs.cloudflare.com/ajax/libs/gsap/3.10.4/gsap.min.js"></script>
    <script src="https://cdnjs.cloudflare.com/ajax/libs/ScrollMagic/2.0.7/ScrollMagic.min.js"></script>
    <script src="https://cdnjs.cloudflare.com/ajax/libs/ScrollMagic/2.0.7/plugins/animation.gsap.min.js"></script>

    <script>
        // GSAP 및 ScrollMagic 설정
        const controller = new ScrollMagic.Controller();

        const tl = gsap.timeline();
        tl.to(".floating", { y: -100, duration: 1, ease: "power1.out" });

        new ScrollMagic.Scene({
            triggerElement: ".floating",
            triggerHook: 0.5,
            duration: "100%"
        })
        .setTween(tl)
        .addTo(controller);
    </script>
</body>
</html>
이 코드는 사용자가 페이지를 스크롤할 때 .floating 클래스가 적용된 요소가 부드럽게 위로 떠오르는 애니메이션을 구현합니다. ScrollMagic.Scene을 사용하여 특정 요소가 뷰포트에 들어올 때 애니메이션이 시작되도록 설정할 수 있습니다.
